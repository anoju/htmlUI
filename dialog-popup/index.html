Dialog 완벽 마스터

시각적 페이드 애니메이션(인터랙션) 통합 및 세 가지 핵심 메서드 실습.

1. showModal() 메서드

배경 영역을 어두운 오버레이로 차단하고 모달 외부 조작을 방지합니다. 폼 입력이나 확인 등 사용자 집중이 강제될 때 사용합니다.

2. show() 메서드

배경 차단 없이 단순 팝업창을 엽니다. 이 버튼을 누르면 팝업이 뜬 상태에서도 화면 스크롤이나 바깥 버튼 클릭이 가능해집니다.

3. close() 메서드

열려있는 모달을 닫고, 열기 직전 위치로 포커스를 센스있게 되돌려 줍니다. 아래 모달 내부의 [닫기] 버튼 코드에 적용되어 있습니다.

4. 부가적인 레이아웃 형태

⚠️ 10년 치 구형 브라우저 실무 지원 시 주의사항

기본적인 <dialog> 기반의 모달은 2022년 상반기(iOS Safari 15.4 이상) 최신 브라우저부터 완벽히 지원됩니다.
만약 2014년~2021년 사이의 구형 아이폰(iOS 14 이하)이나 IE 환경까지 지원해야 한다면 showModal() 호출 시 에러가 발생합니다.

💡 실무 권장 해결책 (Polyfill 사용법):
아래 코드를 HTML 파일과 자바스크립트에 추가하기만 하면, 구형 기기에서도 최신 showModal() 문법이 완벽하게 돌아가게끔 브라우저가 패치됩니다. 
<!-- 1. Polyfill CDN 스크립트 및 CSS 추가 (head 영역에 삽입) -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/dialog-polyfill/0.5.6/dialog-polyfill.min.css">
<script src="https://cdnjs.cloudflare.com/ajax/libs/dialog-polyfill/0.5.6/dialog-polyfill.min.js"></script>

<!-- 2. 자바스크립트에서 다이얼로그 요소 구문 적용 (사용 전 1회 호출) -->
<script>
  const myDialog = document.getElementById('dialogBase');
  dialogPolyfill.registerDialog(myDialog);
</script>

상세 정보 입력

인터랙션 팁: 모달이 열릴 때와 닫힐 때 자바스크립트 'is-closing' 클래스를 활용한 부드러운 열리고 닫히는 애니메이션이 연결되어 있습니다.
