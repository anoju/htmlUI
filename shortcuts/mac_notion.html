Notion

macOS 단축키 가이드

가장 많이 사용되는 단축키

페이지 내 검색 ⌘+F
검색 / 페이지 이동 ⌘+P 또는 ⌘+K
페이지 URL 복사 ⌘+L
뒤로 가기 ⌘+[
앞으로 가기 ⌘+]
다크/라이트 모드 전환 ⌘+⇧+L

데스크톱 탐색

새 페이지 생성 ⌘+N
새 창 열기 ⌘+⇧+N
새 탭 생성 ⌘+T
새 탭으로 열기 ⌘+클릭
새 창으로 열기 ⌥+⇧+클릭

텍스트 서식 (선택 영역에 적용)

굵게 ⌘+B
기울임 ⌘+I
밑줄 ⌘+U
취소선 ⌘+⇧+S
인라인 코드 ⌘+E
링크 추가 ⌘+K

마크다운 스타일 빠른 서식

굵게 **텍스트**
기울임 *텍스트*
취소선 ~텍스트~
인라인 코드 `텍스트`
글머리 기호 목록 * 또는 - 또는 + + Space
번호 매기기 목록 1. + Space
할 일 체크박스 [] + Space
H1 제목 # + Space
H2 제목 ## + Space
H3 제목 ### + Space
토글 목록 > + Space
인용 블록 " + Space
구분선 ---

블록 생성 및 스타일

텍스트 블록 생성 ⌘+⌥+0
H1 제목 생성 ⌘+⌥+1
H2 제목 생성 ⌘+⌥+2
H3 제목 생성 ⌘+⌥+3
할 일 목록 생성 ⌘+⌥+4
글머리 기호 목록 생성 ⌘+⌥+5
번호 매기기 목록 생성 ⌘+⌥+6
토글 생성 ⌘+⌥+7
코드 블록 생성 ⌘+⌥+8
새 페이지 생성 ⌘+⌥+9
확대 ⌘++
축소 ⌘+-

블록 편집 및 선택

현재 블록 선택 Esc
블록 편집 Enter
블록 내 전체 선택 ⌘+A
블록 삭제 Backspace 또는 Delete
블록 복제 ⌘+D
블록 위로 이동 ⌘+⇧+↑
블록 아래로 이동 ⌘+⇧+↓
블록 들여쓰기 Tab
블록 내어쓰기 ⇧+Tab
블록 열기/수정 ⌘+Enter
선택된 블록 편집 ⌘+/
셀 오른쪽으로 채우기 ⌘+R
셀 아래로 채우기 ⌘+D

댓글 및 멘션

댓글 생성 ⌘+⇧+M
사람 멘션 @ + 이름
페이지 멘션 @ + 페이지명
날짜 멘션 @ + 날짜
리마인더 설정 @remind + 날짜

링크 생성

페이지 링크 [[ + 페이지명
하위 페이지 생성 [[ + 하위페이지명, "하위 페이지 추가" 선택
외부 페이지에 링크 [[ + 페이지명, "다음에 새 페이지 추가..." 선택
