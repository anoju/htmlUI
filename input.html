text 타입 : 가장 기본적인 한 줄 텍스트 입력 필드입니다.
text 타입 (maxlength JS 처리) : 자바스크립트를 이용해 최대 글자 수를 제어합니다.
password 타입 : 입력한 내용이 마스킹 처리되어 보이지 않는 비밀번호 필드를 생성합니다.
email 타입 : 이메일 주소를 입력받으며 브라우저 차원에서 기본 유효성 검사를 제공합니다.
tel 타입 : 전화번호를 입력하며 모바일에서 전화번호 전용 키패드를 띄웁니다.
url 타입 : 웹 주소를 입력받으며 URL 형태 여부를 브라우저 자체적으로 검증합니다.
search 타입 : 검색어 입력에 특화되어, 모바일 등에서 전용 키패드가 표시되는 경우가 많습니다.
number 타입 : 숫자를 입력할 수 있고 스피너를 이용해 값을 증감할 수 있습니다.
range 타입 : 슬라이더 형태로 숫자 범위를 지정할 수 있습니다.
date 타입 : 특정 연도, 월, 일을 선택할 수 있는 달력을 제공합니다.
time 타입 : 시, 분 등의 특정 시간을 선택할 수 있게 합니다.
datetime-local 타입 : 지역 시간 기준으로 날짜와 시간을 선택할 수 있게 합니다.
month 타입 : 연도와 월 단위로 날짜를 선택할 수 있게 합니다.
week 타입 : 특정 연도와 그 해의 몇 번째 주차인지를 선택할 수 있습니다.
radio 타입 : 동일 그룹(name 속성) 내에서 하나만 선택 가능한 라디오 버튼입니다.
 
checkbox 타입 : 선택 및 해제가 가능한 체크박스를 생성합니다.
color 타입 : 색상을 선택할 수 있는 위젯을 제공합니다.
file 타입 : 파일을 선택하거나 기기에서 업로드할 수 있도록 돕습니다.
button 타입 : 클릭 가능한 일반 버튼을 생성합니다.
submit 타입 : 작성한 폼 데이터를 서버 등 목적지로 제출합니다.
reset 타입 : 폼 내부에 있는 모든 입력값을 초기값으로 되돌립니다.
image 타입 : 이미지 형태의 폼 제출 버튼을 만듭니다.
hidden 타입 : 사용자에게는 보이지 않지만 폼 전송 시 서버로 데이터가 부가적으로 전달됩니다.
(화면에 표시되지 않음)
