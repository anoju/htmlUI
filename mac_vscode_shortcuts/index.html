macOS 단축키 가이드

일반 편집

줄 잘라내기 (빈 선택) ⌘+X
줄 복사 (빈 선택) ⌘+C
줄 위/아래로 이동 ⌥+↑/↓
줄 위/아래로 복사 ⇧+⌥+↑/↓
줄 삭제 ⇧+⌘+K
아래에 줄 삽입 ⌘+Enter
위에 줄 삽입 ⇧+⌘+Enter
들여쓰기/내어쓰기 ⌘+]/[
줄의 시작으로 이동 Home
줄의 끝으로 이동 End
파일의 시작으로 이동 ⌘+↑
파일의 끝으로 이동 ⌘+↓
줄 접기/펼치기 ⌥+⌘+[/]
모든 줄 접기 ⌘+K ⌘+0
모든 줄 펼치기 ⌘+K ⌘+J
주석 토글 ⌘+/
블록 주석 토글 ⇧+⌥+A
단어 래핑 토글 ⌥+Z

다중 커서 및 선택

커서 삽입 (위) ⌥+⌘+↑
커서 삽입 (아래) ⌥+⌘+↓
실행 취소 마지막 커서 작업 ⌘+U
줄 끝에 커서 삽입 ⇧+⌥+I
현재 선택 찾기 ⌘+F
다음 찾기 선택 ⌘+D
모든 일치 항목 선택 ⌘+⇧+L
모든 일치하는 단어 선택 ⌘+F2
선택 영역 확장 ⇧+⌥+→
선택 영역 축소 ⇧+⌥+←
열(상자) 선택 ⇧+⌥ + 드래그
열(상자) 선택 (위) ⇧+⌥+⌘+↑
열(상자) 선택 (아래) ⇧+⌥+⌘+↓
열(상자) 선택 페이지 위 ⇧+⌥+⌘+PgUp
열(상자) 선택 페이지 아래 ⇧+⌥+⌘+PgDn

검색 및 바꾸기

찾기 ⌘+F
바꾸기 ⌥+⌘+F
다음 찾기/이전 찾기 ⌘+G / ⇧+⌘+G
모든 일치 항목 선택 ⌥+Enter
다음 찾기 선택 추가 ⌘+D
마지막 선택을 다음 찾기로 이동 ⌘+K ⌘+D

다국어 편집

정의로 이동 F12
정의 미리보기 ⌥+F12
Quick Fix ⌘+.
참조 보기 ⇧+F12
기호 이름 바꾸기 F2
공백 문자 바꾸기 ⇧+⌘+L
선택 영역 트림 ⌘+K ⌘+X
언어 모드 변경 ⌘+K M

편집기 관리

편집기 닫기 ⌘+F4, ⌘+W
폴더 닫기 ⌘+K F
편집기 분할 ⌘+\
첫 번째/두 번째/세 번째 편집기 그룹으로 포커스 ⌘+1 / 2 / 3
이전/다음 편집기 그룹으로 포커스 ⌘+K ⌘+←/→
편집기를 왼쪽/오른쪽으로 이동 ⌘+K ⇧+⌘+←/→
활성 편집기 그룹 이동 ⌘+K ←/→

파일 관리

새 파일 ⌘+N
파일 열기 ⌘+O
저장 ⌘+S
다른 이름으로 저장 ⇧+⌘+S
모두 저장 ⌥+⌘+S
닫힌 편집기 다시 열기 ⇧+⌘+T
탭 유지 (미리보기 모드에서) ⌘+K Enter
열기 ⌘+↓
측면에서 열기 ⌘+K ↓

표시

전체 화면 토글 ⌃+⌘+F
편집기 레이아웃 토글 (가로/세로) ⌥+⌘+0
확대 / 축소 ⌘+= / -
사이드바 토글 ⌘+B
탐색기 표시 ⇧+⌘+E
검색 표시 ⇧+⌘+F
소스 컨트롤 표시 ⌃+⇧+G
디버그 표시 ⇧+⌘+D
확장 표시 ⇧+⌘+X
바꾸기 (파일에서) ⇧+⌘+H
탐색기에서 세부 정보 토글 ⇧+⌘+J
출력 표시 ⇧+⌘+U
오류 및 경고 표시 ⇧+⌘+M
통합 터미널 토글 ⌃+`

디버그

시작/계속 F5

단계별 진입/나가기 F11 / ⇧+F11
단계별 실행 F10
호버 표시 ⌘+K ⌘+I

통합 터미널
