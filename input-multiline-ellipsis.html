말줄임(Ellipsis)과 줄바꿈(Multiline) 구현 가이드

Best Practice

스마트 커스텀 Input (Fake Input)

사용자가 지적하신 input의 한계를 극복하는 방법입니다.
평소에는 div로 완벽한 말줄임을 보여주고, 클릭 시에만 실제 입력창으로 전환합니다.

단일 라인 말줄임 + 입력
이 텍스트는 input 태그임에도 불구하고 평상시에 완벽하게 말줄임표가 표시됩니다.
멀티라인 말줄임(2줄) + 입력
멀티라인 말줄임 예시입니다. 클릭하면 전체 내용을 수정할 수 있는 textarea로 변하며, 포커스를 잃으면 다시 2줄 말줄임 상태로 돌아갑니다. 아주 긴 텍스트를 입력해보세요.
Single Line

말줄임표가 있는 Input

포커스가 없을 때 긴 텍스트를 ...으로 처리합니다.
CSS의 text-overflow: ellipsis 속성을 사용합니다.

Multiline

ContentEditable (추천)

div에 contenteditable 속성을 부여하여 내용에 따라 높이가 자동으로 늘어나는 유연한 입력 필드를 만듭니다.

Multiline

자동 높이 조절 Textarea

기존 textarea를 사용하면서 스크립트를 통해 입력 내용에 맞춰 높이를 동적으로 조절합니다.

Display

멀티라인 말줄임 (Clamp)

입력 필드는 아니지만, 긴 본문을 특정 줄 수(예: 3줄)에서 잘라야 할 때 -webkit-line-clamp를 사용합니다.

이 텍스트는 멀티라인 말줄임 예시입니다. 여러 줄의 텍스트가 있을 때 사용자가 읽기 적당한 수준까지만 보여주고 나머지는 말줄임표로 처리하는 디자인 패턴입니다. 주로 게시판 목록이나 카드 디자인에서 본문 요약을 보여줄 때 매우 유용하게 사용됩니다. CSS의 line-clamp 속성을 활용하면 복잡한 자바스크립트 계산 없이도 구현이 가능합니다.
